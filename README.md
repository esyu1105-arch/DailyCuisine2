# Daily Cuisine

A lightweight, installable meal-planning PWA for choosing lunch and dinner ideas with bilingual instructions.

## Features
- Random meal generation for lunch and dinner
- Bilingual UI (Chinese and English)
- Re-roll individual dishes or the full menu
- Progressive Web App support for offline-friendly use

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
